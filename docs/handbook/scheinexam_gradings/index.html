Gradings for Scheinexams

Roles:CorrectorTutor

Overview#

After selecting a scheinexam from the dropdown menu you can see the gradings the students have for the selected exam. To add a grading to or change the grading of a student click the "Enter Points" button on his/her card.

  1. Scheinexam Selection: Select the scheinexam you want to view.

  2. Student's Grading: The current grading of a student.

    note

    If a student has no grading all exercises are displayed with 0 points.

  3. Enter Points: Enter points for the selected scheinexam and student (see below).

Enter Points#

  1. Student Selection: Select the student you want to change the grading of.

    Unsaved changes

    If you have unsaved changes you are prompted to save them before switching. If do not the changes are lost.

  2. Points for exercises: Enter the points the student got for each exercise. Please note that exercises are always "bundled" together and that no sub-exercises are shown.

  3. Save: Save the changes for the currently selected student.

  4. Reset: Reset your changes for the student to the last saved ones.

  5. Go back: Go back to the overview page.
